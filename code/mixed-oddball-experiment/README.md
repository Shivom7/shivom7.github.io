# Mixed audiovisual oddball experiment

A PsychoPy protocol designed for a planned cross-sectional study of how semantic context may influence candidate markers of ageing across people engaged in different meditation practices.

## Protocol components

- five-minute baseline rest;
- audiovisual oddball blocks with standard and rare stimuli;
- event and section markers for physiological-data synchronization;
- working-memory and simple reaction-time tasks;
- randomized task order;
- bilingual English/Hindi participant instructions;
- post-task surveys and DASS-21; and
- timestamped trial, response and marker logging.

## Important status note

This is research-in-development code for an experimental design, not a validated diagnostic or clinical instrument. Stimulus timing, marker synchronization, hardware configuration, translations, consent/ethics materials and survey scoring should be validated in the target laboratory before participant deployment.

## Reproducibility note

The script passes static Python syntax compilation in the portfolio build. Running it requires PsychoPy, a compatible Psychtoolbox audio backend and the intended acquisition hardware.
